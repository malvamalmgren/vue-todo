# Vue Todo App

A small todo application built with Vue 3, TypeScript, Vite, Tailwind CSS, and shadcn-vue/Reka UI components.

## Features

- Login/logout demo flow
- Add and remove tasks
- Set task priority: low, medium, or high
- Sort tasks by date, content, or priority
- Pin important tasks to keep them at the top
- View task creation time from the clock tooltip
- Responsive layout for desktop and mobile

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- shadcn-vue
- Reka UI
- Lucide icons

## Getting Started

```bash
npm install
npm run dev
```

## Notes

This project stores todos in local component state only. Refreshing the page clears the task list.
